# 關聯式資料庫

### 先備知識
*   完成 [Linux 課程](https://linkedin.github.io/school-of-sre/level101/linux_basics/intro/)
*   安裝 Docker（用於實驗部分）

### 這門課程的期望
你將了解什麼是關聯式資料庫、它們的優點，以及一些 MySQL 的特定概念。

### 本課程不涵蓋的內容
*   深入的實作細節

*   進階主題如正規化、分片

*   特定的管理工具

### 介紹
資料庫系統的主要目的是管理資料。這包括資料的存儲、新增資料、刪除不再使用的資料、更新現有資料、在合理反應時間內擷取資料，以及維護系統運作的其他維護任務等。
